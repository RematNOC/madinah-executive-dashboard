# Madinah AI Transformation Executive Dashboard

This is a high-fidelity, bilingual (Arabic/English) executive dashboard presenting the 5-pillar digital transformation strategy for the 30-store jewelry retail chain in Madinah, KSA.

## 🚀 Deployment
This project is designed to be deployed to Vercel. 

## 🛠 Tech Stack
- **Next.js 14** (App Router)
- **Tailwind CSS** (Styling)
- **React** (UI)

## 🇸🇦 Localization
- **RTL (Right-to-Left)** support for Arabic.
- **Language Switcher** on the top right for instant toggle between English and Arabic.
- **Vision 2030** alignment themes.

## 📁 Structure
- `/src/app/page.tsx`: Main bilingual dashboard logic.
- `/src/app/globals.css`: Tailwind and gradient definitions.
